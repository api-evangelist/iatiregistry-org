# IATI Registry (iatiregistry-org)

IATI Registry is a open data portal open-data portal running **CKAN**, exposing a standardized catalog API over ~13,656 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/iatiregistry-org/refs/heads/main/apis.yml)

- **IATI Registry CKAN Action API** — base `https://iatiregistry.org/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://iatiregistry.org)
